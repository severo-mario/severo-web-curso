O que é HTML

HTML (Hypertext Markup Language - Linguagem de Marcação de Hipertexto), como o próprio nome sugere, não pode ser considerada uma linguagem de programação, mas sim como uma linguagem de marcação que informa aos navegadores como eles devem estruturar as páginas web que visitamos.

O HTML consiste basicamente em uma série de elementos que utilizamos para encapsular, delimitar ou marcar as diferentes partes de conteúdos desejadas, de modo que essas informações possam aparecer da maneira que desejarmos. Por exemplo, podemos destacar o que é um título principal, ou dar ênfase a determinado fragmento de texto, criar links que referenciam outras páginas internas ou até mesmo outros sites. Enfim, podemos fazer muitas coisas, semanticamente falando e é isso que começaremos ver a partir de agora. Vamos considerar, por exemplo, a seguinte linha de texto:

Oi, meu nome é Mario Severo.
